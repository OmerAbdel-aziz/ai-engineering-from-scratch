# 05 — Jupyter Notebooks

- JupyterLab installed and running on port 8888
- Understand magic commands: `%timeit` (multi-run average for microbenchmarks) vs `%%time` (single-run wall time for training)
- Familiar with keyboard shortcuts: `Shift+Enter` run cell, `A`/`B` insert above/below, `DD` delete, `M`/`Y` toggle markdown/code
- Understand kernel model: separate Python process, shared state between cells, hidden dependency trap
- Understand "explore in notebooks, ship in scripts" workflow
- Know how to restart kernel and "Run All" to verify linear execution
